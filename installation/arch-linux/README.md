# ARCH LINUX

[Arch Linux](https://wiki.archlinux.org/) je Linux distribucija namjenjena za x86-64 procesorske arhitekture. Instalira se kao minimalni osnovni sustav koji se kasnije onda konfigurira po korisnikovim željama. Konfiguracija sustava se najčeše izvodi preko ljuske. Arch Linux nudi svoj vlastiti upravitelj *pacman*. Arch Linux koristi *rolling release* način rada koji omogućuje konstantno ažuriranje paketa bez reinstalacije cijelog sustava. Ovo omogućuje da sustav bude ažuran.

## Preuzimanje slike

Arch Linux se može preuzeti na mnogo načina, moguće je direktno preuzeti pomoću HTTP-a ili pomoću BitTorrent-a. Moguće je također preuzeti sliku za virtualni stroj ili pak Docker sliku ako ga se ne želi izravno instalirati na sustav. Više stvari o načinima preuzimanja se mogu vidjeti [ovdje](https://archlinux.org/download/).
